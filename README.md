# PoolStuff

Commercial pool products, service, and installation website.

Built with [Next.js](https://nextjs.org) 16, React 19, TypeScript, and Tailwind CSS v4.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Home
- `/products` — Product catalog with search and category filter
- `/new-products` — Recently added products
- `/services` — Dolphin & Lovibond service centre and commercial fit-outs
- `/about` — About PoolStuff
- `/contact` — Contact form

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm start` — run production server
- `npm run lint` — run ESLint
